# HiddenTestSummary

Replaces individual hidden-test rows in student-facing responses. The student learns how many hidden tests they passed and the point impact, but never sees the underlying TestCase names, descriptions, or logs.

## Properties

| Name           | Type   |
| -------------- | ------ |
| `label`        | string |
| `passedCount`  | number |
| `totalCount`   | number |
| `pointsEarned` | number |
| `pointsTotal`  | number |
