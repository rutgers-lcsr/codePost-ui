## ✨ Interaction Flow

```
User Journey:
═══════════════════════════════════════════════════

1. View Header
   ↓
   See file count and required/optional breakdown

2. Read Workflow Banner
   ↓
   Understand: Download → Complete → Submit

3. Scan Table
   ↓
   See existing files, what has code, what's required

4. Add Files (Choose path):

   Path A: Manual Entry
   ↓
   Type directory/filename → Click Add
   ↓
   Upload code for file → View/edit

   OR

   Path B: Bulk Upload
   ↓
   Click upload → Select .zip
   ↓
   All files extracted automatically
   ↓
   View/edit any file as needed

5. Mark Required Files
   ↓
   Check boxes for files students must submit

6. Save Assignment
   ↓
   Click Save on parent dialog
```
