## What codePost can do

### Code annotation & rubrics

Leave inline comments directly on student code. Build reusable rubrics so your whole team gives consistent, calibrated feedback across every submission.

### Automated testing

Run student code automatically against your test suite in a sandboxed environment. Supports Python, Java, R, and more. See [Environment & Testing Ops](/docs/instructor-environment-testing).

### AI-assisted grading

Graders can draft a comment and use AI to refine its tone and clarity. Instructors can generate test scripts from assignment specs. Powered by your choice of OpenAI, Gemini, Ollama, or a custom provider. See the [AI Guide](/docs/ai-guide).

### Anonymous grading

Hide student identities from graders to reduce bias. Graders see anonymous IDs; admins retain full visibility. See [Features Guide](/docs/features).

### Programmatic access

Upload submissions, download grades, and automate workflows via the [Python SDK](/docs/programmatic-comments) or the [REST API](https://codepost-api.cs.rutgers.edu/api/schema/swagger-ui/).
